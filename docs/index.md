## Introduction

* This datasheet describes the design and functionality of the temperature sensing module used in the exploration device. The purpose of this document is to outline the hardware components, operation, and system integration of the module. The temperature sensing module measures the temperature in the area, to then that signal is sent out to the display subsystem.

### Project Summary

* Team 307 is working on a submersible exploration unit that will work in aquatic environments for monitoring and also collecting samples. Sensors, microcontrollers, and communication interfaces are integrated into the system to collect and read environmental data, ranging from temperature, pressure to distance measurements.
* Here is the datasheet to Team 307 [team report.](https://egr-314-team-307-spring-2026.github.io/Team307.github.io/)

### My Contribution

* My contribution to this project is by designing a Temperature sensing subsystem unit. Each component includes the choice of a suitable temperature sensor, system microcontroller integration of the sensor and how hardware must be designed for communication. My subsystem will recieve data from one of my teamates subsystem which then my subsystem will store the data as well as computing temperature sensor values. After, I take the stored data as well as my value data and output the data to the OLED subsystem.
